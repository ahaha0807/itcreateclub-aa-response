```shell
curl itcreate-club.herokuapp.com
```

1. Launch Browser(Chrome | FireFox)（ChromeかFireFoxをきどうしてね）
2. Access to [This URL](http://itcreate-club.herokuapp.com/html)（このURLにあくせすしてね）
